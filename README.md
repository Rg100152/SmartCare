🩺 SmartCare Diagnostics Portal

A modern, responsive, and interactive Device Diagnostics & Support Portal built with HTML, CSS, and Vanilla JavaScript. This project demonstrates a professional dashboard UI with glassmorphism, animations, issue reporting, ticket management, and analytics in a single HTML file.

«Note: This is an educational UI prototype created for learning and portfolio purposes only. It is not affiliated with any real company or service.»

---

✨ Features

- 📊 Interactive Dashboard
- 🔧 Device Diagnostics Simulator
- 📝 Issue Reporting Form
- 🎫 Ticket Generation & Tracking
- 📈 Analytics Dashboard
- 🌙 Dark & Light Mode
- 📱 Fully Responsive Design
- ✨ Glassmorphism UI
- 🎨 Smooth Animations & Transitions
- 🔔 Toast Notifications
- 📈 Animated Statistics
- ⚡ Single File (HTML, CSS & JavaScript)

---

🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome
- Google Fonts (Poppins)

---

🚀 Getting Started

1. Download or clone this repository.
2. Open "index.html" in any modern web browser.
3. No installation or dependencies required.
